# Tylor Good Portfolio

Precision machining and manufacturing portfolio website.

## Deploy
Push to GitHub and deploy via Render (or any static host).

## Structure
- `index.html` — Homepage with project cards
- `gallery.html` — Category gallery with lightbox
- `data.json` — Manifest of all images and PDFs
- `images/` — Category image folders
- `pdfs/` — Technical drawings
- `css/style.css` — FreeCodeCamp-inspired styling
- `js/main.js` — Gallery loading and lightbox
